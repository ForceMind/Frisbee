# Frisbee

Frisbee 是一个静态网页项目。

## 项目简介

本项目包含一个 `index.html` 文件，适用于静态页面展示、前端练习或快速原型开发。

## 使用方法

1. 克隆仓库：
   ```sh
   git clone https://github.com/ForceMind/Frisbee.git
   ```
2. 进入项目目录：
   ```sh
   cd Frisbee
   ```
3. 直接用浏览器打开 `index.html` 文件即可预览页面。
